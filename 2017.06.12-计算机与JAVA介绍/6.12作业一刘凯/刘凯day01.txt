1.快捷键和DOS命令的练习。
快捷键：
	Ctrl+A	全选
	Ctrl+C	复制
	Ctrl+V	粘贴
	Ctrl+X	剪切
	Ctrl+Z	撤销
	Ctrl+S	保存
DOS命令：
	d: 回车	盘符切换
	dir(directory):列出当前目录下的文件以及文件夹
	md (make directory) : 创建目录
	rd (remove directory): 删除目录
	cd (change directory)改变指定目录(进入指定目录)
	cd.. : 退回到上一级目录
	cd\: 退回到根目录
	del (delete): 删除文件,删除一堆后缀名一样的文件*.txt
	cls : (clear screen)清屏
	exit : 退出dos命令行

5.几个小问题?
	A:JDK,JRE,JVM分别是什么?

		JDK:JDK(Java Development Kit) 是 Java 语言的软件开发工具包。它包含了JAVA的运行环境，JAVA工具和JAVA基础的类库。
		JRE:JRE为Java Runtime Environment的简称，它包括Java虚拟机（jvm）、Java核心类库和支持文件。它不包含开发工具(JDK)--编译器、调试器和其它工具。
		JVM:JVM是Java Virtual Machine（Java虚拟机）的缩写

	B:Java语言是跨平台的吗?JVM是跨平台的吗?
		
		JAVA语言是跨平台的，JVM不是跨平台的。
	
6.什么是注释?注释分为几种?注释有什么作用?
	注释：用于解释说明程序的文字
	Java中注释分类格式：	单行注释 格式： //注释单行文字
				多行注释 格式： /*  注释多行文字  */
				文档注释 格式：/** 注释文档文字 */
